## @jarvis/config

**`@jarvis/config`** es el paquete de configuración del ecosistema **`J.A.R.V.I.S.`**

Este package será responsable de cargar, normalizar y exponer configuración para aplicaciones que usen el runtime de **`J.A.R.V.I.S.`**

---

## Objetivo

Permitir que una app pueda definir su configuración mediante archivos como:

```
settings.json
.env
```
